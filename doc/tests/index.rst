==============
The Test Suite
==============

*Erbsland Configuration Language* comes with an extensive test suite and test tools which allow you to test any parser implementation thoughtfully. If you like to have your parser listed in the implementations chapter, your parser must pass 100% of all tests for your parser-tier and the features your parser announces.
